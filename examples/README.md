# Examples

This directory contains minimal example scenarios for `inner-alignment-companion`.

## Purpose

These files are here to help a reader quickly understand:

- what kinds of user inputs fit this skill
- how the response structure usually works
- what kind of tone and reframing the skill aims for

## Important note

These are examples, not rigid scripts.
A good use of the skill should still adapt to:

- the user's actual emotional state
- the degree of urgency
- whether the issue is emotional, relational, structural, or professional
- whether the right move is reflection, rest, boundary, expression, or outside support
