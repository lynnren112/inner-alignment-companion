# Example 2 — 关系里长期委屈但舍不得离开

## User Input

我知道这段关系让我很委屈，可我就是舍不得离开。我也说不上来为什么，明明已经很累了，但一想到真的离开就更难受。

## Example Response Structure

### 1. 你现在真正卡住的点
你卡住的可能不是“要不要离开”本身，而是你把“离开”内在地等同成了“我会失去爱、失去连接、失去被需要的感觉”。

### 2. 事实 vs 定义
- 事实：这段关系长期让你委屈、消耗、很累。
- 定义：如果我离开，我可能再也遇不到；我是不是太狠；是不是我不够会经营关系。

### 3. 被触发的核心信念
可能被激活的是：
- 我需要忍耐，才配拥有关系
- 我一设边界，就会失去爱

### 4. 这件事在反馈你什么
这段关系也许正在反馈你：你已经太习惯用委屈自己来换取连接。
问题未必只是“对方对你怎样”，也包括你如何一次次绕过自己的边界。

### 5. 更对齐的理解方式
离开一段持续消耗你的关系，不等于你失败了，
更可能意味着你终于不再把“忍耐”误认为“爱”。

### 6. 今天可以做的一小步
今天先不逼自己做最终决定。
只列出两张清单：
- 我在这段关系里持续压下去的真实感受是什么
- 我真正需要但一直没有被认真对待的边界是什么
