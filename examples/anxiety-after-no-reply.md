# Example 1 — 对方不回消息后的焦虑

## User Input

他两天没回我消息，我整个人都很乱。我知道不一定是大事，但就是会忍不住脑补，是不是我不重要、是不是他其实没那么在乎我。

## Example Response Structure

### 1. 你现在真正卡住的点
你卡住的可能不只是“他没回”，而是这件事触碰到了你很深的“我是不是不够重要”的感觉。

### 2. 事实 vs 定义
- 事实：他两天没有回复你。
- 定义：我不重要；他不在乎我；这段关系可能没希望。

### 3. 被触发的核心信念
这里可能被激活的是：
- 我必须被及时回应，才说明我值得被在乎
- 一旦对方沉默，就说明我不够重要

### 4. 这件事在反馈你什么
这次触发在反馈你：你仍然很容易把“被回应”当成价值确认。
情绪不是在证明你不值得，而是在提醒你，这个确认系统还放在别人手里。

### 5. 更对齐的理解方式
对方没回，首先只说明“对方暂时没回”。
它还不足以自动证明你不重要。
真正值得被看见的是：为什么你的内在系统会这么快把沉默翻译成自我否定？

### 6. 今天可以做的一小步
今天先不追着要答案，也不急着压情绪。
只写下来一句：**“如果我不是在害怕自己不重要，我会怎么理解这次沉默？”**
