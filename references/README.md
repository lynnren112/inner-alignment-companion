# References & Orientation

本目录用于说明 `inner-alignment-companion` 的来源、边界与使用取向。

## 1. 来源不是“照搬”，而是“抽取共同内核”

这个 skill 并不是对某一本书、某一位老师或某一套灵性体系的机械复述，
而是尝试从以下来源中抽取可用于现实陪伴的稳定结构：

- 《太傻天书》：识破幻象、停止把社会剧本当成唯一真实
- 《与神对话》：从恐惧叙事转向创造性与主体性
- 《你值得过更好的生活》：配得感、允许、从低配自我叙事中毕业
- 巴夏（Bashar）体系：定义、信念、情绪反馈、状态与对齐行动

## 2. 为什么要做这个 skill

很多人在痛苦里，并不是缺“道理”，而是缺一个更清晰的拆解结构。

这个 skill 的价值不在于：

- 告诉人“该怎么活”
- 用高维语言压住具体痛苦
- 快速给出一个漂亮结论

而在于帮助人：

- 先被接住
- 区分事实与定义
- 看见触发背后的旧信念
- 重新回到主体性
- 找到一个不过度消耗的下一步

## 3. 它不是什么

这个 skill 不是：

- 心理治疗替代品
- 危机干预工具
- 医疗建议工具
- “万事都是你吸引来的”解释器
- 用灵性话术合理化压迫、创伤或结构性问题的工具

## 4. 推荐的使用姿势

更适合这样用：

- 当你已经被某件事触发，但不想只停留在情绪打转
- 当你隐约知道自己又进入旧模式了，但看不清结构
- 当你想在“现实感”与“高视角理解”之间找到平衡
- 当你需要一个轻量但有穿透力的陪伴框架

## 5. 不推荐的使用姿势

不推荐这样用：

- 把它当作“替我做决定”的工具
- 把它当作“证明一切都没问题”的安慰剂
- 在急性危机中继续做抽象灵性解读
- 用它替代现实中的边界、沟通、休息、求助与专业支持

## 6. 维护建议

如果你长期维护这个 skill，建议后续持续补充：

- 更多高频场景示例
- 更细的边界案例
- 不同语气版本（温柔版 / 直指版 / 极简版）
- 面向不同 agent 系统的安装说明
